# To-Do List App

## Description

The To-Do List App is a Python-based task management application designed to help users organize and manage their daily activities efficiently. The application allows users to add new tasks, view existing tasks, and remove completed tasks through a simple menu-driven command-line interface.

This project demonstrates fundamental Python programming concepts such as lists, loops, conditional statements, user input handling, and basic data management. It serves as a beginner-friendly productivity tool and provides practical experience in developing interactive command-line applications.

## Features

* Add new tasks
* View all tasks
* Remove completed tasks
* Menu-driven interface
* Simple and user-friendly design

## Technologies Used

* Python 3

## How to Run

1. Open the project in PyCharm.
2. Run the `main.py` file.
3. Select an option from the menu.
4. Add, view, or remove tasks as needed.
